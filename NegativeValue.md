# 正の数・負の数

2025.05.10 - takeshi.23.jan

## はじめに

高校以降”純理系”人間だったものの、「マイナス掛けるマイナスはプラス」で中学数学に躓いた苦い経験があります。改めて振り返ると、負の数は

- どのような背景か?
- どのようなルールか?
- 複雑な計算を如何にこなすか?

が混ざり合っているように感じます。

テストで高得点を取る為には複雑な計算をこなす練習が欠かせませんが、一方、背景自体に違和感を持ったまま計算をこなすだけなのは健全な教育とは言えない、とも思います。

本小論考では特に背景の点を丁寧に見直すことを主眼としています。

### 導入


小学校では足し算・引き算を習いました。例えば

- $6 - 3 = 3$
- $7 - 3 = 4$

のような計算です。また大きな数や小数、分数なども習いましたね。ただし**引き算の結果は常に正**という暗黙の了解がありました。これは


- 小学校算数の引き算：”引かれる数”よりも”引く数”の方が小さくなければならない

という制約をかけていた、とも言えます。


一方、中学校では「方程式」という新しい方法を学びます。これは問題文中にある「わからない量」を文字$x$に置き換え、それを「解く($x$ = ***の形に式変形する)」事で答えを半自動的に求めることが出来る、スゴイ方法なんだけど、その際「引き算の答えが正でないとダメ」というのは強い制約になります。何故なら引き算が出てくる度に「どっちが大きいか」を気にしなければならないから

*数学の歴史的には17世紀位までは負の数は広く認められていなかったら、毎回場合分けしていました。しかし一部の人が負の数を認めた計算で同じ答えをより早く求められることに気づき、徐々に負の数が広く認められるようになりました。現代では負の数を教えて、それに基づく方程式の解き方を学ぶ方が圧倒的に効率が良いことがわかっているので、中学１年の最初で負の数が教えられる、という訳なんです。

そこで方程式を導入する前に、

- 中学校の引き算：”引かれる数”と”引く数”は任意

という約束をまずします。つまり中学数学では小学算数で扱ってこなかった

- $3 - 6$
- $3 - 7$

のような計算もOK、とする訳なんだ。そして$6 = 3 + 3$や$7 = 3 + 4$と書けることに注意すると

$$
3 - (6) = 3 - (3 + 3) \\
        = 3 - 3 - 3 \\
        = (3 - 3) - 3 \\
        = -3 
$$

$$
3 - (7) = 3 - (3 + 4) \\
        = 3 - 3 - 4 \\
        = (3 - 3) - 4 \\
        = -4
$$

と計算した結果も認めることにします。


まとめると

- 小学校算数では引き算は”引かれる数”よりも”引く数”の方が小さくなければならない
- 中学数学では”引かれる数”と”引く数”に制約はない

事から負の数が導入されます。


### 負の数の演算

負の数を導入しただけでは(残念ながら)話は終わりません。方程式を自由に解く為には負の数の演算、つまり

- 和: 足し算
- 差: 引き算
- 積: 掛け算
- 商: 割り算

を導入する必要があります。この内「負の数の掛け算」が特に厄介な所なので、順を追って説明していくね。


#### 負の数の足し算1

まず負の数の足し算だけど、小学校の算数でも

$$
3 + 3 - 6 = 0
$$

と計算できるのは良いよね?ここで上の式を次のように見直します。

$$
3 + (3 - 6) = 0
$$

さっきやったように

$$
(3 - 6) = (-3)
$$

なので、

$$
3 + (-3) = 0
$$

になります。この式から逆に

- 負の数の足し算 : 小学算数で習った引き算と同じ

とわかります。


#### 負の数の足し算2

先に進む前に負の数の足し算について、一つ要請をしておきましょう。正の数の足し算では「足し算は順番を変えても結果が変わらない」事が成り立っていました。

$3 + 6 = 6 + 3$
$5 + 11 = 11 + 5$

同じ事を負の数の足し算にも要請します。つまり**正の数だけでなく、負の数の足し算も順番を変えてよい**と要請します。例えば

$3 + (-6) = (-6) + 3$

を認める、ということです。

注意して欲しいのは正の数は現実世界に対応づいた概念だけど、負の数は方程式を簡単に解きたいという人間(正確には数学者・物理学者・エンジニアetc)の願望から生まれたものです。なので**目的(方程式が簡単に解ける)に叶ってさえいれば、新しい要求をするのも自由**ということなんだ(もちろん他の人を説得できるだけの十分な根拠を持つ必要はあるけれども)。


#### 負の数の引き算

小学校算数で次のような計算を習ったのは覚えているかな?


- $1-1 = 0$
- $3-3 = 0$
- $1.534-1.534 = 0$

つまり元の数から同じ数を引くと0になるわけだけど、**負の数にも同じ性質を要求します**。

つまり

- $(-3) - (-3) = 0$
- $(-1.534) - (-1.534) = 0$

と要請する、ということです。

ところで

- $(-3) + 3 = 0$

だったよね?
- $(-3) - (-3) = 0$
- $(-3) + 3 = 0$

を比較すると

- $-(-3) = 3$

が導かれます。つまりマイナスが2個重なるとプラスになる、訳です。


### 負の数の掛け算1

今度は負の数の掛け算を考えていくね。その為に次の計算式を考えます。

- $2 * 0 = 0$

ところで

- $0 = (-3) + 3$

なので、

- $2 * ((-3) + 3) = 0$

と書けます。さらに左辺のかっこを外すと

- $2 * ((-3) + 3) = 2 * (-3) + 2 * 3 = 0$

です。ところで

- $= -(2 * 3) + 2 * 3 = 0$

でもあるので、

$2 * (-3) = - (2 * 3)$

であることがわかります。


### 負の数の掛け算2

先と同じように

- $0 * 3 = 0$

という式を考えます。ところで


$2 + (-2) = 0$

なので、

- $0 * 3 = (2 + (-2)) * 3 = 2 * 3 + (-2) * 3 = 0$

と書けます。一方

- $2 * 3 + (-(2 * 3)) = 0$

でもあるので、

- $(-2) * 3 = -(2 * 3)$

であることがわかります。



### 負の数の掛け算3

- $(-2) * 0 = -(2 * 0) = 0$

であることに注目します。

- $0 = (-3) + 3$

なので、

$0 = (-2) * ((-3) + 3)$

と書けます。右辺をさらに展開すると

$0 = (-2) * ((-3) + 3) = (-2) * (-3) + (-2) * 3$

- $(-2) * 3 = -(2 * 3)$

なので、


$0 = (-2) * (-3) - (2 * 3)$

が言えます。これから

$(-2) * (-3) = (2*3)$

が導かれます。つまり「マイナス掛けるマイナスはプラス」な訳です
